# Задание 2 — Проектирование API

**Описание:**  
Интернет-магазин "Петрушка Зеленая" расширяется. В мобильном приложении создается новый экран, который показывает магазины-партнёры.

## Эндпоинт API

**GET** `/v1/partner-stores`  

**Описание:** Возвращает список партнерских магазинов для мобильного приложения.

**Пример запроса:**

```bash
curl -X GET "https://api.petrushka-green.com/v1/partner-stores" \
     -H "Accept: application/json"
