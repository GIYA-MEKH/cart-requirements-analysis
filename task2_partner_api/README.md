# Задание 2 — Проектирование API

**Описание:**  
Интернет-магазин "Петрушка Зеленая" расширяется, и в мобильном приложении создается новый экран, который показывает магазины партнеров.

## Эндпоинт API

**GET** `/v1/partner-stores`

**Описание:**  
Возвращает список партнерских магазинов для мобильного приложения.

## Пример запроса

```bash
curl -X GET "https://api.petrushka-green.com/v1/partner-stores" \
     -H "Accept: application/json"

Пример ответа

Ответ находится в файле partner_stores_response.json

Пояснения к полям JSON
id — уникальный идентификатор магазина
name — название магазина
logo_url — ссылка на логотип магазина
rating — средний рейтинг
category — категория товаров
website_url — внешний сайт, куда переходит пользователь при клике
